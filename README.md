# Random Beach

> Shortcut um zufällige Folgen der Drei ??? aufzurufen und den Fortschritt zu tracken

Dieser Shortcut nutzt Daten die von [drei-metadaten](https://api.dreimetadaten.de/) zur Verfügung gestellt werden. Das zugehörige Repository findest du [hier](https://github.com/YourMJK/dreimetadaten).


### Installation

`Random Beach.shortcut` aus `/bin/` Ordner herunterladen und mit Doppelklicken öffnen.


### Verwendung

Beim normalen Ausführen des Shortcuts wird nach einmaligem Onboarding eine zufällige Folge auf der zuletzt verwendeten Plattform (zur Zeit Spotify order Apple Music) geöffnet. Jedes Öffnen einer Folge wird als Play getrackt und die zufällige Auswahl bezieht sich immer auf die Folgen mit der niedrigsten Anzahl an Plays.


Über das Teilen-Menü kann der Shortcut ebenfalls ausgeführt werden. Hier kannst du Folgen welche du gerade so hören möchstest auch tracken lassen. Verwendest du hier eine andere Plattform als beim Mal davor änderst du so auch deine Plattform für die Zufallswiedergabe.
