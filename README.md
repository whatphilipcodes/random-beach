# Random Beach

> Kurzbefehl, um zufällige Folgen der Drei ??? aufzurufen und den Fortschritt zu tracken

Dieser Shortcut nutzt Daten, die von [drei-metadaten](https://api.dreimetadaten.de/) zur Verfügung gestellt werden. Das zugehörige Repository findest du [hier](https://github.com/YourMJK/dreimetadaten). Für die Verwendung benötigst du ein `Apple-Gerät` sowie ein aktives Premium-Abonnement bei `Spotify` oder `Apple Music`.


### Installation

> Kurzbefehl [hier](https://www.icloud.com/shortcuts/b90df4667e644514b3c19a8181c6778f) herunterladen

Das Gerät sollte dich automatisch durch den Import leiten. Danach kann der Kurzbefehl wie jeder andere verwendet werden.


### Verwendung

Beim normalen Ausführen des Kurzbefehls wird nach einmaligem Onboarding eine zufällige Folge auf der zuletzt verwendeten Plattform (zurzeit Spotify oder Apple Music) geöffnet. Jedes Öffnen einer Folge wird als Play getrackt, und die zufällige Auswahl bezieht sich immer auf die Folgen mit der niedrigsten Anzahl an Plays.


Über das Teilen-Menü kann der Shortcut ebenfalls ausgeführt werden. Hier kannst du Folgen, die du gerade so hören möchtest, ebenfalls tracken lassen. Verwendest du hier eine andere Plattform als beim Mal davor, änderst du so auch deine Standard-Plattform für die Zufallswiedergabe.
